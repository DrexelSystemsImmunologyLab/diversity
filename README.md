Diversity
=========

Gregory W. Schwartz

```
Usage: diversity [-l|--inputLabel LABEL] [-r|--inputOrder ORDER] [-w|--inputWindow WINDOW] [-i|--inputFasta FILE] [-n|--removeN] [-o|--output FILE]
  Return the diversity at each position for all sequences in a fasta file

Available options:
  -h,--help                Show this help text
  -l,--inputLabel LABEL    The label for this particular dataset
  -r,--inputOrder ORDER    The order of true diversity
  -w,--inputWindow WINDOW  The length of the sliding window for generating fragments
  -i,--inputFasta FILE     The fasta file containing the germlines and clones
  -n,--removeN             Remove 'N' and 'n' characters
  -o,--output FILE         The csv file containing the diversities at each position
```
